# 보리네 말해줘 개인정보처리방침

보리네 말해줘(이하 "앱")는 사용자의 개인정보를 소중하게 보호하며, 관련 법령을 준수합니다.  
본 방침은 앱에서 어떤 정보가 수집·이용되는지와 사용자의 선택권을 설명합니다.

## 1. 수집·이용하는 정보

### 1) 사용자가 직접 생성하는 데이터
- 화면 캡처 이미지(사용자가 명시적으로 요청한 경우에만)
- 형광펜 표시 영역, 인식 결과 텍스트

위 데이터는 **기기 내에서만 처리**되며, 서버로 전송되지 않습니다.

### 2) 필수 권한
- **오버레이 권한**: 플로팅 토글 표시
- **화면 캡처 권한**: 사용자가 요청한 화면을 캡처

권한은 기능 제공을 위한 목적에만 사용됩니다.

### 3) 광고 및 분석
현재 앱은 **광고/분석 SDK를 사용하지 않습니다.**

## 2. 이용 목적
- 사용자가 선택한 화면 영역의 텍스트 인식(OCR)
- 인식된 텍스트의 음성 읽기(TTS)
- 플로팅 토글 및 캡처 기능 제공

## 3. 보관 및 삭제
- 캡처 이미지와 인식 결과는 **로컬 저장(임시)** 되며, 앱을 종료하거나 캐시가 정리되면 삭제됩니다.
- 사용자는 앱을 삭제함으로써 관련 데이터를 완전히 삭제할 수 있습니다.

## 4. 이용자의 권리
사용자는 기기 설정에서 권한을 언제든지 허용/차단할 수 있습니다.

## 5. 방침 변경
방침이 변경될 경우, 공개된 정책 URL을 통해 최신 내용을 안내합니다.
