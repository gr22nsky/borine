# 🌾 보리네 말해줘

### 화면 글씨 읽기 도우미 앱

### Expo + React Native / Android 전용

---

## 1. 앱 개요

보리네 말해줘는 글씨가 잘 보이지 않는 시니어를 위해
**현재 화면에서 사용자가 직접 짚은 부분만 읽어주는 보조 도구 앱**이다.

이 앱은 기존 앱을 대체하지 않으며,
"지금 보고 있는 화면이 무슨 뜻인지"를 이해하도록 돕는 역할만 수행한다.

---

## 2. 해결하려는 핵심 문제

- 작은 글씨, 회색 글씨가 잘 보이지 않음
- 팝업·알림의 의미를 이해하지 못함
- 글자 확대·접근성 설정까지 도달하지 못함
- 화면 전체를 읽어주는 앱은 시끄럽고 부담됨

👉 해결 원칙:
**자동 판단 ❌ / 사용자가 직접 짚은 부분만 읽기 ⭕**

---

## 3. 핵심 UX 컨셉

- 항상 떠 있는 **플로팅 토글**
- 화면을 **정지 캡처**
- **형광펜으로 자유롭게 긋기**
- 손을 떼면 자동으로 인식 준비
- 읽기 / 취소 버튼으로 명확한 제어

---

## 4. 전체 UX 흐름

1. 플로팅 토글 ON
2. 다른 앱 사용 중 토글 터치
3. 현재 화면 정지 캡처
4. 형광펜으로 읽을 부분 표시
5. **손을 떼는 순간**:

   - 첫 읽기인 경우에만
     - 중앙 안내: "잠시만 기다려주세요"
     - 하단 배너 광고 3초 노출
6. 읽기 버튼 표시
7. 읽기 버튼 터치 → TTS 재생
8. 같은 화면에서 재읽기 시 광고 미노출
9. 취소 버튼 → 즉시 원래 화면 복귀

---

## 5. 주요 기능 상세

### 5.1 플로팅 토글

- 항상 다른 앱 위에 표시
- 손으로 잡아 자유롭게 이동 가능
- 클릭과 이동은 제스처로 구분
- Foreground Service로 유지

---

### 5.2 화면 캡처

- MediaProjection 사용
- 실시간 화면 ❌
- 항상 정지 화면만 처리

---

### 5.3 형광펜 입력

- 사각형 선택 ❌
- 자유 드로잉 방식
- 기본 굵기 매우 두껍게
- 전체 초기화 버튼 제공

---

### 5.4 OCR 처리

- 형광펜 영역만 마스킹 OCR
- 전체 화면 OCR ❌
- 온디바이스 OCR 우선

---

### 5.5 음성 읽기(TTS)

- OS 내장 TTS 사용(무료)
- 자동 재생 ❌
- 버튼으로만 재생
- 같은 텍스트 여러 번 재생 가능

---

## 6. 광고 UX 정책

### 기본 원칙

- 읽는 중 광고 ❌
- 기능을 가리지 않음
- 자동 재생 / 소리 ❌

### 노출 규칙

- **첫 읽기 1회에만** 노출
- 중앙 안내 문구:

  > "잠시만 기다려주세요"

- 하단 배너 광고 1개
- 3초 후 자동 종료

---

## 7. 기술 스택 및 구조

- Expo + React Native
- TypeScript
- Android 전용

### 주요 Android 요소

- MediaProjection (화면 캡처)
- SYSTEM_ALERT_WINDOW (오버레이)
- Foreground Service

---

## 8. 상태 관리 개념

- `isPremium`: 유료 여부
- `hasShownAdForThisCapture`: 동일 캡처 내 광고 노출 여부
- `ocrResult`: 인식된 텍스트

---

## 9. 제한 사항

- 은행/보안/DRM 앱은 캡처 제한 가능
- 일부 화면은 텍스트 인식 정확도 낮을 수 있음

---

## 10. MVP 범위

- 플로팅 토글
- 캡처 + 형광펜
- OCR + TTS
- 읽기 / 취소 버튼
- 1회 결제 유료 모델

---
