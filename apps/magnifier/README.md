# 🔍 보리네 돋보기

### 시니어 맞춤형 초간편 디지털 돋보기 앱

### Expo + React Native / Android 전용

---

## 1. 앱 개요

보리네 돋보기는 노안이나 저시력으로 인해 일상생활의 작은 글씨(약병, 메뉴판, 영수증 등)를 보기 힘든 시니어를 위한 **초간편 디지털 돋보기 앱**이다.

복잡한 카메라 기능은 모두 배제하고, **'크게 보기'**와 **'선명하게 보기'**라는 본질에만 집중하여 설계되었다.

---

## 2. 해결하려는 핵심 문제

- 스마트폰 기본 카메라 앱의 UI가 너무 작고 복잡함 (셔터, 필터, 모드 전환 등)
- 손떨림으로 인해 작은 글씨에 초점을 맞추기 어려움
- 어두운 식당이나 밤중에 메뉴판/약병 글씨가 보이지 않음
- 고대비가 필요한 저시력 사용자를 위한 모드가 부재함

👉 해결 원칙:
**복잡한 설정 ❌ / 실행 즉시 크게 보기 ⭕ / 흔들려도 멈춰서 보기 ⭕**

---

## 3. 핵심 UX 컨셉

- **실행 즉시 돋보기:** 앱을 켜자마자 바로 카메라 화면이 나타남
- **초대형 슬라이더:** 화면 하단에 손가락으로 밀기 쉬운 거대 줌 슬라이더 배치
- **화면 정지(Freeze):** 셔터 소리 없이 화면을 그대로 멈춰서 자세히 볼 수 있는 기능
- **자동 조명:** 어두운 곳을 감지하거나 필요 시 원터치로 손전등 켜기
- **시인성 강화 필터:** 흑백 모드, 색상 반전 모드 제공

---

## 4. 주요 기능 (MVP)

1. **실시간 확대/축소 (Zoom)**
   - 1배율부터 최대 배율까지 부드러운 확대
   - 두 손가락 핀치 줌 + 하단 대형 슬라이더 동시 지원

2. **화면 일시정지 (Freeze Frame)**
   - 사진으로 저장하지 않고 화면만 멈춤 (갤러리 용량 차지 방지)
   - 멈춘 상태에서도 확대/축소 및 필터 적용 가능

3. **손전등 제어 (Flashlight)**
   - 원터치로 카메라 플래시 ON/OFF
   - 어두운 환경에서 가독성 극대화

4. **시각 보조 필터 (Visual Filters)**
   - 일반 모드 (컬러)
   - 고대비 흑백 모드 (글자 강조)
   - 색상 반전 모드 (눈부심 방지 및 가독성 향상)

---

## 5. 기술 스택 및 라이브러리

- **Framework:** Expo (SDK 52+)
- **Camera:** `expo-camera` (카메라 피드 및 줌 제어)
- **UI/UX:** `react-native-reanimated`, `lucide-react-native` (아이콘)
- **Storage:** `expo-file-system` (필요 시 스냅샷 임시 저장)

---

## 6. 향후 확장 계획

- **텍스트 추출(OCR):** 돋보기로 보고 있는 글자를 음성으로 읽어주는 기능 연동
- **이미지 저장:** 정말 중요한 정보는 갤러리에 바로 저장하는 기능
- **QR 코드 인식:** 시니어용 대형 QR 코드 스캐너 모드 추가
