# Play Console 스토어 등록 문구 (보리네 알리미)

## 앱 이름

보리네 알리미

## 짧은 설명 (최대 80자)

오늘 할 일과 반복 일정을 간단히 관리하고, 필요한 시간에만 알림으로 챙겨주는 앱.

## 자세한 설명 (최대 4000자)

보리네 알리미는 "오늘 할 일"을 빠르게 기록하고, 완료/미완료를 체크하면서 하루를 정리할 수 있는 간단한 일정 관리 앱입니다.

### 주요 기능

- 오늘 할 일: 날짜별 할 일 목록 확인 및 완료/미완료 체크
- 분류: 아침/점심/저녁으로 할 일을 나눠 보기
- 반복 일정: 매일/매주 반복과 기간(시작/종료) 설정
- 캘린더: 날짜별 할 일과 완료 현황 확인
- 기록 공유: 선택한 기간 기록을 텍스트로 공유
- 알림: 설정한 시간에 미완료 할 일이 있을 때만 알림

### 개인정보 및 데이터 처리 안내

- 회원가입/로그인 없음
- 사용자가 입력한 할 일/기록/설정은 기본적으로 기기(로컬)에 저장
- 광고(AdMob) 포함 시, 광고 제공/측정을 위해 광고 식별자(Advertising ID) 등 광고 관련 정보가 처리될 수 있음
