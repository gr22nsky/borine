# Play Console 스토어 등록 문구 (보리네 말해줘)

## 앱 이름

보리네 말해줘

## 짧은 설명 (최대 80자)

필요한 글씨만 직접 짚어 읽어주는 시니어 친화 화면 읽기 도우미.

## 자세한 설명 (최대 4000자)

보리네 말해줘는 글씨가 잘 보이지 않는 시니어를 위해, 사용자가 직접 짚은 부분만 읽어주는 화면 글씨 읽기 도우미입니다.

### 주요 기능

- 플로팅 토글: 언제 어디서나 화면을 캡처할 수 있는 간편한 버튼
- 형광펜 표시: 읽고 싶은 영역만 직접 짚어서 표시
- 텍스트 인식(OCR): 표시된 영역의 글씨를 정확하게 인식
- 음성 읽기(TTS): 인식된 텍스트를 선명한 음성으로 읽어주기

### 개인정보 및 데이터 처리 안내

- 회원가입/로그인 없음
- 캡처 및 인식 결과는 서버 전송 없이 기기(로컬) 내에서만 처리
- 광고(AdMob) 포함 시, 광고 제공/측정을 위해 광고 식별자(Advertising ID) 등 광고 관련 정보가 처리될 수 있음
